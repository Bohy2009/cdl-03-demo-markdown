# cdl-03-demo-markdown

Create a program that prints "Hello, World!"

Also create a `Makefile` to build the program.

## Starting Point

Start from the following skeleton:
```c
#include <stdio.h>

int main(void){
    printf("CATS AND DOGS");
    ...
}
```
## Test
To test, run:
```conslole
./helloworld | grep `Hello, World!`
```

## Build
Build the program using:
```
Make

```
